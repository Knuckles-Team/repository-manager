---
name: git-status
description: Checks status for an individual project. Use for working status of a project.
---

### Overview
Project status via `git_action`.

### Key Tools
- `git_action`: "cd  "<repositories_directory>"/"<project>" && git status"

### Usage Instructions
1. Status: "git status".

### Examples
- Status: `git_action` with command="git status" directory="projects directory" project="the project (directory name) inside projects directory to run command in"

### Error Handling
- Specify the project the user is trying to work with inside the repositories directory.
